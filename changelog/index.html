v2.1.15 September 7, 2025

📦 New Features

  • The toolbar has been revamped: Text descriptions have been added to buttons to avoid user confusion; some function buttons have been moved to the search bar, so the search bar no longer supports collapsing.
  • When clicking a group in the tab bar, the last tab within that group will be focused by default. This feature is optional.
  • Added monitoring for tab attachment, detachment, and movement from windows, with the list updated synchronously.
  • A "How to use" entry has been added to the navigation bar and user center.

👍 Optimizations

  • Import examples have been added for quick import rules, including format instructions for import rules that were missing in previous versions.
  • Optimizations have been made at the development level; component splitting has been implemented to enhance the extension's performance.

🎉 Bug Fixes

  • Fixed: For tabs not grouped by rules, a highlight background color will now be displayed when hovering the mouse (previously missing).

v2.1.10 August 15, 2025

📦 New Features

  • Added an option: When a group in the top tab bar is expanded, automatically focus on the first tab in the group.（Auto grouping must be enabled）
  • Added an option for handling tabs outside automatic grouping rules; now you can choose to auto-group tabs that match the rules and not group those that don't. (Auto grouping rules must exist)

👍 Optimizations

  • Added an entry in the settings panel to adjust the sidepanel's position (left/right) in the browser.
  • When users modify shortcuts, the changes will be synchronously displayed in the settings panel.

🎉 Bug Fixes

  • Fixed: Closing the first tab causes the tab group title to disappear.
  • Fixed: When restoring a tab, it incorrectly jumps to a newly opened tab.
  • Fixed: Remove the duplicate icon of the "Clear Rules" button in the settings panel.
