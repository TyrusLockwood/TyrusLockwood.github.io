How to use it?

Click the extension icon in the top right corner of the browser. Pin it to the extensions bar.

Pin it to the extensions bar.

You can click on Tabbiy in the extensions bar to display the list.

Use the shortcut to open the list more easily. (ALT+D).

Of course, you can customize this shortcut.

Core

Click "Auto" at the bottom left of the list, and the browser will organize all tabs into groups by domain.

Newly opened tabs will auto into relevant groups.

Of course, you can also customize the grouping rules in the settings panel.

Clicking the in the list or toolbar will temporarily save the tabs.

You can restore tabs when you need to access them again, just like OneTab.

You can switch between recently opened tabs using a shortcut.

If you frequently need to compare multiple tabs, this feature seems tailor-made for you.（Recently tab）

The shortcut (ALT+E) has been preset for you.

Clicking the in the list or toolbar will hibernate tabs to save memory.

When you need to revisit these tabs, you can wake them up.
