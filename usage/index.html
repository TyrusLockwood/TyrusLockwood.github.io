How to use it?

Click the extension icon in the top right corner of the browser. Pin it to the extensions bar.

Pin it to the extensions bar.

You can click on Tabbiy in the extensions bar to display the popup.

Use the shortcut to open the popup more easily. (ALT+D)

Of course, you can customize this shortcut.

Core

Click "Auto Group" at the bottom left of the popup, and the browser will organize all tabs into groups by domain.

Newly opened tabs will auto into relevant groups.

Of course, you can also customize the grouping rules in the settings interface.

You can switch between recently opened tabs using a shortcut.

If you frequently need to compare multiple tabs, this feature seems tailor-made for you.（Recently tab）

The shortcut (SHIFT+ALT+D) has been preset for you.
