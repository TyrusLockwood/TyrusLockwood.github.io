How to use it?

Click the extension icon in the top right corner of the browser , find Tabbiy and pin it to the extensions bar.

At this point, you can click on Tabbiy in the extensions bar to display the list popup.

You can also use the shortcut to open the popup window more conveniently. The shortcut (ALT+D) has been preset for you.
Of course, you can customize this shortcut.(Open the popup)

Core

1. Click 'Auto Group' at the bottom left of the popup list, and the browser will organize all tabs into groups by domain.

Of course, you can also customize the grouping rules in the settings interface.

2. You can switch between recently opened tabs using a shortcut key. If you frequently need to compare multiple tabs, this feature seems tailor-made for you.（Recently tab）
The shortcut (ALT+D) has been preset for you.

Of course, you can customize this shortcut.（Recently tab）
